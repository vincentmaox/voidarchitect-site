# voidarchitect-site

老茅（虚空建筑师 / Void Architect）的工作室主页。

- 域名: https://www.voidarchitect.studio
- 部署: Vercel（沿用旧 voidcompass 项目，切换 git 来源）
- 数据源: 构建期 fetch `_ProjectOS/data/public-registry.json`
- 技术栈: Astro 5 + React islands + Tailwind 4 + Framer Motion + R3F (WebGL Hero)

## 运作

注册为 ProjectOS 子项目跟踪。三频共振 4/4/5 = 13/15 → All-in。
72h MVA 起始 2026-06-15。

## 命令

```sh
npm run dev      # localhost:4321
npm run build    # 构建到 ./dist/
npm run preview  # 本地预览 build
```
