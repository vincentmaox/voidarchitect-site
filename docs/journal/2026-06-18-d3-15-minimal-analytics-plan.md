# 2026-06-18 — D3.15 最小事件埋点方案

**日期：** 2026-06-18  
**主体：** 老茅 + 老赫  
**阶段：** D3.15 传播反馈准备

## 决策

D3.15 只做 analytics 方案，不立刻接平台。原因是站点刚进入可传播状态，当前更需要明确“看什么信号”，而不是马上引入复杂统计、cookie、数据库或广告追踪。

## 输出

新增文档：

- `docs/analytics-plan.md`

## 事件设计

保留 4 类高信号事件：

1. `homepage_product_click`
   - 首页产品卡点击
   - 判断哪个产品最吸引第一次访问者

2. `product_cta_click`
   - 产品详情页咨询 CTA 点击
   - 判断哪个产品有真实咨询意图

3. `github_click`
   - GitHub 链接点击
   - 判断用户是否需要代码可信度背书

4. `language_switch_click`
   - 中英文切换
   - 判断英文入口是否有真实传播价值

## 平台建议

优先 Plausible：

- 轻量
- 隐私友好
- 自定义事件简单

备选 Vercel Analytics：

- 集成简单
- 更适合页面级访问统计

## 暂不做

- 数据库记录
- 用户画像
- session replay
- 热力图
- 广告像素
- 跨站追踪
